# Portfolio Muhammad Dava Kayla Kalam Perdana

Website portfolio pribadi untuk menampilkan profil, tools, proyek, detail studi kasus, CV, dan kontak profesional.

## Fokus Portfolio

- Web Development
- UI/UX Design
- Decision-support systems
- Image processing
- Sistem informasi berbasis web

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Remix Icon
- AOS

## Fitur

- Hero section dengan CTA untuk download CV, kontak, dan daftar proyek.
- Kartu proyek dengan tipe proyek, stack teknologi, link detail, live demo, GitHub, dan YouTube demo.
- Halaman detail proyek berisi masalah, peran, fitur utama, hasil, stack, dan galeri.
- Navbar responsive dengan hamburger menu di mobile.
- Form kontak berbasis mailto.
- Metadata SEO dan Open Graph.

## Menjalankan Project

```bash
npm install
npm run dev
```

## Build Production

```bash
npm run build
```
